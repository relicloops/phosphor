Phosphor Documentation
=======================

Phosphor is a pure C CLI tool for the NeonSignal/Cathode ecosystem. It
scaffolds projects from template manifests, builds, cleans, and runs
diagnostics. The name references the phosphor coating inside neon tubes --
the layer that converts UV discharge into visible light.

CLI contract: ``phosphor create``, ``phosphor build``, ``phosphor clean``,
``phosphor doctor``, ``phosphor version``, ``phosphor help``.
